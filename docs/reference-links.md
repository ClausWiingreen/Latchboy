# Game Boy Public Reference Links

This document tracks publicly available technical references and ROM test suites
used during emulator development.

See also: [`docs/rom-usage-policy.md`](./rom-usage-policy.md) for repository
rules on acceptable ROM/test assets.

## Public Hardware Documentation

- **Pan Docs** (community-maintained Game Boy technical reference)  
  https://gbdev.io/pandocs/
- **Game Boy CPU Manual (archival scan)**  
  https://realboyemulator.files.wordpress.com/2013/01/gbcpuman.pdf
- **GBDev resources index**  
  https://gbdev.io/resources.html

## Public Test ROM Suites

- **Blargg Game Boy CPU instruction tests**  
  https://github.com/retrio/gb-test-roms
- **Mooneye GB test suite**  
  https://github.com/Gekkio/mooneye-test-suite
- **mealybug-tearoom-tests** (PPU behavior reference)  
  https://github.com/mattcurrie/mealybug-tearoom-tests

## Usage Notes

- Use only legally obtained ROMs for gameplay and compatibility testing.
- Prefer public-domain/homebrew test ROMs for automated CI and regression checks.
