# ROM Usage Policy

This project does **not** distribute commercial Game Boy ROMs, BIOS dumps, or
copyrighted game assets.

## What is acceptable for this repository

- Homebrew and public-domain ROMs.
- ROMs that are explicitly licensed for redistribution by their authors.
- Public emulator test ROMs/suites that are published for compatibility testing.

## What is not acceptable for this repository

- Commercial game ROM binaries.
- Nintendo boot ROM dumps/firmware blobs with unclear or restrictive licensing.
- Links that promote piracy or unauthorized downloads of copyrighted games.

## Contributor guidelines

- Do not commit copyrighted ROM images to this repository.
- Keep automated CI/regression validation focused on public test ROM suites.
- If a ROM or test asset has unclear licensing, open an issue before adding it.
